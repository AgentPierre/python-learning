# Automate the Boring Stuff with Python

Code from [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/) by Al Sweigart.

## Book Overview

This book teaches practical programming for total beginners. Topics include:
- Python basics
- Pattern matching with regular expressions
- Reading and writing files
- Organizing files
- Web scraping
- Working with Excel, Word, and PDF files
- Email and text messages
- GUI automation

## Chapter Organization

- **ch01_basics/** - Python fundamentals
- **ch06_strings/** - String manipulation and pattern matching
- **ch08_files/** - Reading/writing files, organizing folders
- **ch11_web_scraping/** - Downloading and parsing web data
- **ch12_excel/** - Reading and editing Excel spreadsheets
- **ch16_email/** - Sending emails and text messages
