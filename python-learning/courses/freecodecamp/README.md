# freeCodeCamp Python Course

Code from the [Scientific Computing with Python](https://www.freecodecamp.org/learn/scientific-computing-with-python/) certification.

## Course Overview

This certification covers:
- Python fundamentals
- Data structures
- Object-oriented programming
- Algorithms and problem solving
- Scientific computing with Python

## Projects

The certification includes 5 main projects:
1. Arithmetic Formatter
2. Time Calculator
3. Budget App
4. Polygon Area Calculator
5. Probability Calculator
